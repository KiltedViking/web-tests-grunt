External links (YUI)

On my site (kiltedviking.net) I add target attribute to certain links to open pages in new windows, so that my page's HTML is (or validates as) correct. Carets (>) are also appended to link to show that link opens a new window.

The "only" thing I need to do is:

  • add a reference to YUI Library with a script tag
  • add script to page or, even better, a reference to script file with script
  • add class attribute to links that should open in new window.

<script src="http://yui.yahooapis.com/3.4.1/build/yui/yui-min.js"></script>

YUI().use('node', function (Y) {
  //Get nodes for external links
  var extLinks = Y.all(".externallink");

  //For each link - add target attribute and add carets
  extLinks.each(function(lnk) {
    lnk.set("target", "_blank");
    lnk.append(" >>");
  });
});
