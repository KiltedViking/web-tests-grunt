Grid Basics

Terms

Container
Parent element and container for items
Items
Child elements positioned in the grid
Cell
Area
One or more cells
Row (track)
Column (track)
Gap
Line
A border between cells. Can be negative
Explicit
Something defined by developer
Implicit
Something left to browser to handle

Tips

  • Styling must be done on items/elements.
  • Grid doesn't exist in HTML/DOM - only in CSS.
  • Don't use percent!
