Basic MUI CSS Components A kitchen sink example

This examples shows off basic components in front-end framework MUI CSS. It seems as if MUI CSS mostly focuses on typography and styling, i.e. styling like that of Bootstrap or Foundation requires a bit more classes.

Containers

There are two kinds of containers in MUI CSS:

  • "fixed" (.mui-container) - widths based on the "four widths": large (lg), medium (md), small (sm), and extra small (xs).
  • fluid (.mui-container-fluid)

Glyphicons

Buttons

Buttons can be created using both button and a tags.

Link (anchor) button

Navs

There are no classes for navs, but they can be created by combining classes for lists with other classes.

Navbars/Appbars

There are no classes for navbars, or appbars as they are called in MUI CSS, but .mui-appbar adds som colour and padding. Example on their site suggests using a table to create an appbar.

Some content of appbar A button A button

Pagination

There are no classes for pagination.

Labels

There are no classes for labels, if not form labels.

Badges

There are no classes for badges.

Page headers

There are no classes for page headers, but there is a few classes for typography.

Alerts

There are no classes for alerts, but if one wants one can use an overlay that prevents scrolling.
