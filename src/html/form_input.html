Form: input - HTML

Test of new types for tag input. Browsers seems to have implemented support for them differently... i.e. a date input might be shown as an ordinary textbox or as a 'drop-down selector'.

New type values

  • search - search box
  • tel - telephone number
  • url - web address
  • email - E-mail address
  • date - date
  • time - time
  • number - numeric value
  • range - numeric value using slider
  • color - colour well

New attributes

  • maxlength - max number of allowed charachters
  • minlength - min number of allowed charachters
  • size - number of visible characters
  • readonly - if contents is read-only
  • required - if value is required
  • multiple - if multiple values are allowed
  • pattern - regular expression to validate value against
  • min - min value allowed
  • max - max value allowed
  • step - granularity when stepping up/down
  • list - ID of list to use as options (use with tag datalist)
  • placeholder - an example value to show if no value entered
