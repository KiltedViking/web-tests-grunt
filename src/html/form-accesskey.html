Form accesskey

Example showing how to add and use accesskey attribute of input controls.

Please note that Firefox users on Windows need to use Shift key as well, i.e. Shift+Alt+<key>, and Mac users need to use Control key as well, i.e. Ctrl+Alt+<key>!
