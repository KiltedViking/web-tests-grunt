Moment.js

Examples showing how to use JavaScript library Moment.js to manipulate and query Date objects.

Create Date object from string

var date1 = moment("2017-09-04")

Adding a month

Date object will be mutated when adding!

date1.add(1, "month")

Clone object before adding to leave original Date object as is. date1 was mutated when 1 month was added above.

date1.clone().add(1, "month")

date1

Expressing how long date is from now

date1.fromNow())
