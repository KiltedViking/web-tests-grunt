WYSIWYG editor

About

Adding WYSIWYG editor to page, TinyMCE's jQuery version here.

Steps taken

  • Add textarea to page in form (and maybe set its ID ["myeditor"]).
  • Add reference to jQuery and TinyMCE source files (if using jQuery version of TinyMCE).
  • Add call to tinymce.init() function to initialise and configure editor, mainly set selector to add editor to ("#myeditor").
  • (In my case I decided to remove the menubar.)
