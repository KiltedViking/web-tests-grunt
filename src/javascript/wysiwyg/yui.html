WYSIWYG editor - YUI

About

Adding WYSIWYG editor to page, YUI here. In version 3 of YUI there is no GUI... (i.e. example mothballed).

Steps taken

  • Add a block element, like div, to page (and maybe set its ID ["myeditor"]).
  • Add reference to YUI source files.
  • Load plug-in "editor-base".
  • Create an instance of EditorBase and call function render() on instance supplying ID of element as parameter.
