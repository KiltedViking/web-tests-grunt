WYSIWYG editor - NicEdit

About

Adding WYSIWYG editor to page, NicEdit here.

Steps taken

  • Add textarea to page in form (and maybe set its ID ["myeditor"]).
  • Add reference to NicEdit source files.
  • Add call to bkLib.onDomLoaded() with anaonymous function as parameter.
  • In anonymous function, create an instance of nicEditor prototype passing value for property iconsPath and calling function panelInstance() passing name of textarea.
