Adding WYSIWYG editor to page, CKEditor with jQuery here that is also inline (not textarea).

Steps taken

  • Add a block element, like div, to page (and maybe set its ID ["myeditor"]).
  • Add attribute "contenteditable" with value "true" to element.
  • Add reference to jQuery and CKEditor source files plus CKEditor's jQuery adapter.
  • Call function ckeditor() on element.
  • (... and click text to edit. ;-))
