One-way binding

This example shows how a textbox and two placeholders (in heading and paragraph) can be connected to an Angular model and when content in textbox changes, so will any other bindings.

Hello world, {{name}}!

Hope you enjoy you visit to my website, {{name}}!

Enter your name:

Please note that this example "pollutes" the global namespace! Solution to this is to add a controller, i.e. wrap model ins a module.
