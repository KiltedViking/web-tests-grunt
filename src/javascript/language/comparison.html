Comparison (== vs. ===) - JavaScript

Values from textboxes are returned as strings and to make comparison easier, comparison of values of different types is allowed, i.e. "5" == 5. This is fine in many cases, but in other cases this becomes troublesome, which is why the strict equality operators (=== and !==) were added to JavaScript, i.e. "5" !== 5.

Equality (==)

If data types don't match, one or both values will be converted.

    Strict Equality (===)

    Types must be the same, i.e. no conversion perfomed.
