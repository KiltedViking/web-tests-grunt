About

These examples show how to use jQuery to:

  • Run code when document has loaded. ($(document).ready(function() {...}) or $(function() {...})
  • Locate elements. ($(<selector>))
  • Get and set content of element. (.text() and .html())
  • Get and set content of form elements. (.val())
  • Get and set attributes. (.attr())
  • Add and trigger event handlers. (.focus() and .<event>())
